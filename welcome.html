Site-Specific Activation

If you want the extension to work only on specific websites, please do the following:

  • Right-click on the extension icon in the browser toolbar:
  • From the menu, select 'Manage extensions'.
  • Select the websites where you want the extension to be automatically enabled. Or just select all websites at once. Sometimes after the updates this option will be disabled, so this is a good troubleshooting step if extension is not activated

Note: When updating the extension, Chrome may prompt for permission for the extension to access websites. If you click 'no', the extension may stop working.

Queue Functionality: Send Prompts Sequentially

OneClickPrompts now includes a powerful Queue feature, allowing you to send multiple prompts one after another with a set delay between them. This is perfect for example when you want AI to search for news and tell you updates every 2 hours.

How to use the Queue:

  1. Open the Floating Panel: The queue controls are located in the floating panel. Click the 'üîº' button next to the prompt input area to open it.
  2. Enable Queue Mode: In the floating panel, find the "Queue" section and toggle on "Enable Queue Mode".
  3. Add Prompts to the Queue: Once enabled, any custom prompt button you click will be added to the queue instead of being sent immediately. You will see icons for queued prompts appear.
  4. Configure the Delay: Set the time delay (in minutes or seconds) between each prompt being sent.
  5. Control the Queue: Use the ‚ñ∂Ô∏è (Play), ‚è∏Ô∏è (Pause), and üîÑ (Reset) buttons to manage the queue.

Important: Please ensure that using the queue feature to send prompts automatically does not violate the Terms of Service of the website you are using. You will be asked to confirm that you have permission from the website owner before using the feature for the first time.

Cross-Chat Prompt Sharing: Reuse Prompts Anywhere
