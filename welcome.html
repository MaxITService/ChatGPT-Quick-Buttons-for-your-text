Welcome to OneClickPrompts!

Thank you for installing OneClickPrompts

This extension helps you save time by adding customizable prompt buttons to popular AI chat interfaces.

Currently supported platforms:

  • ChatGPT
  • Claude
  • Microsoft Copilot
  • DeepSeek
  • Google AI Studio
  • Grok

Getting Started

1

Visit any supported AI chat platform

Go to ChatGPT, Claude, Copilot, or any other supported platform.

2

Look for the OneClickPrompts buttons

You'll see a row of buttons above the chat input area.

3

Click a button to insert a prompt

Click any button to instantly insert that prompt into the chat.

4

Customize your buttons

Click the extension icon in your browser toolbar to open the settings panel where you can customize buttons and create profiles.
