Pinned Extension Button

Important: You must pin the extension button. To do this, click on your extension menu, then select 'Pin' near the icon of this extension.

Once pinned, when you click the extension button, you will see a menu where you can edit the text for all the buttons that you add.

In this UI, you can fully control the extension and, at the bottom, import/export profiles.

Site-Specific Activation

If you want the extension to work only on specific websites, please do the following:

  • Right-click on the extension icon in the browser toolbar.
  • From the menu, select 'Manage extensions'.
  • Select the websites where you want the extension to be automatically enabled.

Note: When updating the extension, Chrome may prompt for permission. If you click 'no', the extension may stop working.
